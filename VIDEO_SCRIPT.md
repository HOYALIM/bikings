## 🎬 촬영 순서 (30초 타임라인)

### **0-5초: 소개 + 전체 화면**
- **말하기**: "안녕하세요, Lab 7 지도 시각화 프로젝트를 소개합니다."
- **화면**: 전체 페이지 보여주기 (제목, 슬라이더, 지도, 범례)
- **액션**: 없음

### **5-12초: 기본 시각화 설명**
- **말하기**: "보스턴 지역의 자전거 교통량을 인터랙티브하게 시각화했습니다. 초록색 선은 자전거 도로를, 원은 BlueBikes 정거장을 나타냅니다."
- **화면**: 지도 확대해서 자전거 도로(초록색 선)와 정거장(원) 보여주기
- **액션**: 마우스로 지도 팬/줌 (선택적)

### **12-18초: 시각화 인코딩 설명**
- **말하기**: "원의 크기는 교통량을, 색상은 출발과 도착의 비율을 보여줍니다."
- **화면**: 큰 원과 작은 원, 파란색/주황색 원들을 가리키며 설명
- **액션**: 마우스로 다른 크기/색상의 원들을 가리키기

### **18-24초: 시간 필터 데모**
- **말하기**: "시간 슬라이더로 하루 중 다른 시간대의 패턴을 탐색할 수 있습니다."
- **화면**: 슬라이더를 움직여서 원들이 크기와 색상이 변하는 것 보여주기
- **액션**: 
  - 슬라이더를 아침 시간(예: 08:00)으로 이동 → 원들이 변함
  - 슬라이더를 저녁 시간(예: 18:00)으로 이동 → 원들이 다시 변함
  - 빠르게 2-3번 이동

### **24-30초: 흥미로운 발견 + 툴팁**
- **말하기**: "가장 흥미로웠던 점은 아침에는 도심으로, 저녁에는 도심에서 나오는 명확한 교통 흐름 패턴을 발견한 것입니다."
- **화면**: 
  - 아침 시간(08:00)으로 설정 → 도심 지역의 원들이 주황색(도착 많음)으로 변함
  - 저녁 시간(18:00)으로 설정 → 도심 지역의 원들이 파란색(출발 많음)으로 변함
  - 마우스로 큰 원 하나에 호버 → 툴팁 보여주기
- **액션**: 
  - 슬라이더 이동
  - 마우스 호버로 툴팁 표시

## 🎯 필수로 보여줘야 할 인터랙션

1. ✅ **지도 팬/줌** (선택적, 시간이 남으면)
2. ✅ **시간 슬라이더 이동** (필수) - 아침/저녁 시간대 보여주기
3. ✅ **마우스 호버 툴팁** (필수) - 정거장 정보 표시
4. ✅ **원 크기/색상 변화** - 슬라이더로 시간 변경 시

## 💡 촬영 팁

1. **화면 녹화**: 전체 브라우저 창 또는 특정 영역만
2. **마우스 커서**: 명확하게 보이도록 (필요시 커서 하이라이트)
